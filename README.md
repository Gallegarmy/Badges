## ➕ Como sumar unha medalla comunidade?


1. **Forkear este repo**

2. **Deseñar unha medalla**  
   Preferentemente cun deseño que distinga a data ou o mes do evento, especialmente no caso de eventos recorrentes.

3. **Poñer nome a unha medalla**
    O nome debe incluír o nome do evento, a comunidade, o mes e o ano, separados por "_" por exemplo:
    ```
        "Caseta_Comunidade_Marzo_2026.jpg"
    ``` 

4. **Facer commit ao repo forkeado**

5. **Facer unha pr ao repo principal**

6. **Contactar cos admins do proxecto**  
   - 📧 Correo: `sysarmygalicia@gmail.com`  
   - 💬 Telegram: `@Qrow01`

Unha action de github subirá a medalla á DB cando a pr sexa mergeada.
